Finger exercise to get back to Flutter


# book_worm

inside of .env folder create a json file for each environment `{ENV}.json` and fill it as such 
```
{
  "RAPID_API_KEY": YOUR_KEY,
  "RAPID_API_HOST": API_HOST,
}

```
to run it 
`flutter run`
